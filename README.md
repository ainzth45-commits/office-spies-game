# 🕵️ เกมที่ 1 — "สายลับในออฟฟิศ" (Office Spies)

เกมกระตุ้นยอดขายแนว **สืบสวนซ่อนบทบาท (social deduction)** สำหรับทีม CRM
เล่นต่อเนื่อง 1 สัปดาห์ ใช้ "ยอดขาย → เหรียญจริง" เป็นพลังในการเล่นเกมการเมือง โหวต ซื้อไอเทม และปั่นเกม

> เอกสารชุดนี้คือ **brief สำหรับ codex** ใช้สร้างทั้งแอปและภาพประกอบ

## ⚡ สรุปแก่นเกมใน 30 วินาที

- ทีม ~11 คน มีสายลับแฝง **2 คน (A & B)** สุ่มลับสุดๆ สปายเห็นหน้ากันเอง
- ขายของ → รับ **เหรียญจริง** ที่ซุป ตามโปรแต่ละวัน (แอดมินยืดหยุ่นเอง)
- เหรียญใช้: **เปิดโหวตกำจัด · ซื้อไอเทม · หมุนกาชา · ซื้อไอเทมโหวตลับ**
- ทีมต้อง **จับสายลับให้ครบ 2 คน** ก่อนหมดสัปดาห์ / สปายต้อง **เอาตัวรอด + ปั่นเกม**
- เริ่ม **จันทร์** จบ **จันทร์หน้าบ่าย** (ข้ามเสาร์-อาทิตย์) ≈ 6 วันเล่นจริง

## 🎯 หลักการออกแบบสำคัญ

1. **ไม่มีใครถูกเตะออกจากเกม** — โดนจับ = เปิดโปง/เสียเปรียบ แต่ทุกคนยังขาย ยังเล่นถึงวันสุดท้าย (กันคนหมดไฟ)
2. **เหรียญเป็นของจริง 100%** — ซุปกำกับดูแล แอปไม่นับยอดเหรียญรายคน (แอป = เกมมาสเตอร์ + ตัวสุ่ม + จอประกาศ)
3. **ตัวเลขเศรษฐกิจทั้งหมดปรับได้** — อยู่ในหน้า config ของแอดมิน ไม่ต้องแก้โค้ด
4. **รันบนเครื่องเดียว ออฟไลน์ได้** — state ต้องอยู่ครบ 1 สัปดาห์ ห้ามหาย (persist + backup)
